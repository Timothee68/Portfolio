Projet de fin d'année Site de yoga

Pour mon projet de fin de formation, j'ai conçu un site web qui comprend un système de réservation et de paiement en ligne.

L'objectif a été de créer un site pour une professeure de yoga afin d'augmenter sa visibilité et accroître sa clientèle. A ce jour, le projet n'est pas terminé mais je continue à travailler dessus pour le finaliser.

Les fonctionnalités principales mises en place sont :

  • Système de réservation
  • Newsletter
  • Blog
  • Vidéo séance de yoga
  • Livre d'or
  • E-commerce
  • Page de contact

Le projet a été codé en Symfony PHP en Programmation Orienté Objet sous le modèle MVP.

En premier lieu, j'ai commencé par faire le maquettage grâce a Figma, puis j'ai fait la modélisation de données grâce a Looping en faisant le MCD / MLD grâce à la méthode Mersie pour concevoir la Base De Données et aider à la création des Entités.

J'ai bien entendu travaillé le référencement naturel SEO durant tout le projet et veillé à la sécurité tout au long de ce processus.

Durée de développement : environ 3 mois.

Technologies utilisé

Langage HTML5 / CSS3 / JS / BOOTSTRAP / JQUERRY / AJAX / PHP / SYMFONY / SQL/DQL

Bundle CKEditor / DomPDF / FullCalendar / EasyAdmin

Api STRIPE pour le paiement en ligne

Projet Forum

Projet réalisé en PHP-7 avec un framework maison sous le modèle MVP (Modèle Vue Présentateur), grâce au système de routing via un index.php.

Le but ici était de recréer un forum. J'ai décidé de prendre le thème du jeu vidéo.

La réalisation de la modélisation de données du MCD/MLD a été faite en amont du projet.

Une fois inscrit sur le forum, l'utilisateur a la possibilité de créer/modifier/supprimer son topic et de répondre à celui des autres membres.

Un administrateur ou un modérateur peut bannir un utilisateur et fermer un topic en plus des droits utilisateurs.

Le travail sur ce projet comprenait également la sécurité et la gestion du CRUD ; la suppression d'un compte amène à l'anonymisation de celui-çi.

Durée de développement : environ 2 semaines.

Technologies utilisé

Langage HTML5 / CSS3 / JS / BOOTSTRAP / PHP / SQL

JQUERRY cdn DataTable.js

Projet Cinéma

Projet réalisé en PHP-7 avec un framework maison sous le modèle MVP (Modèle Vue Présentateur), grâce au système de routing via un index.php.

Le but ici était de recréer un site dans l'esprit d'AlloCiné.

La réalisation de la modélisation de données du MCD/MLD a été faite en amont du projet.

Le projet comprend la gestion du CRUD ainsi que la liaison de plusieurs relations entre les entités créées, ce qui permet de pouvoir naviguer entre le film, son réalisateur et ses acteurs principaux.

Il amène aussi la possibilité de faire une recherche par nom de film, d'acteur ou de réalisateur via une barre de recherche.

Le site permet d'ajouter/modifier/suppimer les catégories suivantes :

  • Film
  • Réalisteur
  • Acteurs
  • Catégories

Durée de développement : environ 2 semaines.

Technologies utilisé

Langage HTML5 / CSS3 / JavaScript / BOOTSTRAP / JQUERRY / PHP / SQL

Projet Formation en Symfony

Projet consistant à reproduire une interface de gestion d'un centre de formation.

L'idée est de gérer les rôles utilisateurs entre formateur/secrétaire/administrateur, il y a différentes parties.

  • Interface de gestion secrétaire
  • Interface de communication entre les formateurs
  • Interface pour voir les Stagiaire (infos)
  • Interface pour voir les sessions de formations

On peut modifier/crée/modifier/supprimer une session avec tout ce que cela comprend, Nom de session/catégorie de session.

On peut ajouter ou enlever un stagiaire d'une session.

On peut à la création d'une session ajouter des modules dynamiquement avec JavaScript.

Durée de développement : environ 2 semaines.

Technologies utilisés

Langage CSS3 / HTML5 / JavaScript / jQuery / PHP / SQL

Framework Symfony / Bootstrap

cdn DataTable / CKEditor
