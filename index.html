Projet de fin d'année Site de yoga

Le projet de fin d'année est un site de yoga avec un Système de réservation et un système de paiement en ligne. Durée 3 mois de travaille

Le but est de crée un site pour une personne de ma famille pour augmenter sa visibilité et sa clientèles. Le projet n'est pas aboutit mais je continue à travailler dessus pour le finalisé

Les fonctionnalité principale mise en place sont :

Système de réservation / Newsletter / Blog / Vidéo séance de yoga / Livre d'or / E-commerce / page de contact.

Le projet à été coder en Symfony PHP en Programation Orienter Objet.

En premier lieux j'ai commencer par faire une maquette complète grâce a Figma, puis j'ai fait la modélisation de données grâce a Looping en faisant le MCD, MLD pour préparer le terrain a la construction du site.

j'ai bien entendue travailler le référencement naturelle SEO durant tout le projet.

La sécurité à été tout au long du procésus un point important sur lequel j'ai travailler.

Technologies utilisé

Langage HTML5 / CSS3 / JS / BOOTSTRAP / JQUERRY / AJAX / PHP / SYMFONY / SQL/DQL

Bundle CKEditor / DomPDF / FullCalendar / EasyAdmin

Api STRIPE pour le paiement en ligne
